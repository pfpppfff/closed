## 支持的数据类型

### 基本数据类型
- `bool` - 布尔值（位操作）
- `byte` - 字节（8位）
- `short` - 短整型（16位）
- `int` - 整型（32位）
- `float` - 单精度浮点数（32位）
- `double` - 双精度浮点数（64位）
- `ushort` - 无符号短整型（16位）
- `uint` - 无符号整型（32位）

### 数据类型大小映射
- `bool`: 1位（按字节计算）
- `byte`: 1字节
- `short/ushort`: 2字节
- `int/uint/float`: 4字节
- `double`: 8字节

## 使用示例对比

### 读取操作示例
```csharp
var plcManager = MultiPlcServiceManager.Instance;

// 读取操作
var qBits = await plcManager.ReadQBitsAsync(0, 0, 8);        // Q0.0-Q0.7
var iBits = await plcManager.ReadIBitsAsync(1, 3, 5);       // I1.3-I2.0
var mInts = await plcManager.ReadMAsync<int>(10, 2);        // MD10, MD14
var mBits = await plcManager.ReadMBitsAsync(0, 0, 16);      // M0.0-M1.7
```

### 写入操作示例
```csharp
// 写入操作
bool[] qValues = { true, false, true, true, false, false, true, false };
var qSuccess = await plcManager.WriteQBitsAsync(0, 0, qValues);  // Q0.0-Q0.7

var qBitSuccess = await plcManager.WriteQBitAsync(1, 5, true);   // Q1.5 = true

int[] mInts = { 100000, 200000 };
var mSuccess = await plcManager.WriteMAsync<int>(10, mInts);     // MD10, MD14

var mSingleSuccess = await plcManager.WriteMSingleAsync<float>(20, 3.14f); // MD20 = 3.14

bool[] mBits = { true, false, true, true, false, false, true, false };
var mBitSuccess = await plcManager.WriteMBitsAsync(0, 0, mBits); // M0.0-M0.7
```

## 地址映射规则

### Q点地址映射
- `WriteQBitsAsync(0, 0, 8位数据)` → Q0.0 到 Q0.7
- `WriteQBitsAsync(1, 3, 5位数据)` → Q1.3 到 Q2.0 (跨字节)
- `WriteQBitAsync(2, 5, value)` → Q2.5

### I点地址映射
- `ReadIBitsAsync(0, 0, 8)` → I0.0 到 I0.7
- `ReadIBitsAsync(2, 1, 12)` → I2.1 到 I3.4 (跨字节)

### M区地址映射
- `WriteMAsync<byte>(0, 4个值)` → MB0, MB1, MB2, MB3
- `WriteMAsync<short>(10, 2个值)` → MW10, MW12
- `WriteMAsync<int>(20, 2个值)` → MD20, MD24
- `WriteMSingleAsync<float>(30, 值)` → MD30
- `WriteMBitsAsync(0, 0, 8位数据)` → M0.0 到 M0.7
