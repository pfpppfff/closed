# S7NET 扩展读取功能使用说明

## 新增功能概述

按照您的要求，我已经添加了以下新的读取功能：

1. **读取Q点（输出点）bool量** - 支持设置长度
2. **读取I点（输入点）bool量** - 支持设置长度  
3. **读取M区不同类型数据** - 支持多种数据类型

## 功能详细说明

### 1. 读取Q点（输出点）bool量

```csharp
/// <summary>
/// 读取Q点（输出点）bool量
/// </summary>
/// <param name="startByte">起始字节</param>
/// <param name="startBit">起始位</param>
/// <param name="length">读取长度（位数）</param>
/// <param name="plcId">PLC ID（可选）</param>
/// <returns>bool数组</returns>
Task<bool[]> ReadQBitsAsync(int startByte, int startBit, int length, string plcId = null)
```

**使用示例：**
```csharp
// 读取Q0.0开始的8个位
var qBits = await plcManager.ReadQBitsAsync(0, 0, 8);

// 读取Q1.3开始的5个位
var qBits2 = await plcManager.ReadQBitsAsync(1, 3, 5);

// 指定PLC读取
var qBits3 = await plcManager.ReadQBitsAsync(0, 0, 16, "PLC1");
```

### 2. 读取I点（输入点）bool量

```csharp
/// <summary>
/// 读取I点（输入点）bool量
/// </summary>
/// <param name="startByte">起始字节</param>
/// <param name="startBit">起始位</param>
/// <param name="length">读取长度（位数）</param>
/// <param name="plcId">PLC ID（可选）</param>
/// <returns>bool数组</returns>
Task<bool[]> ReadIBitsAsync(int startByte, int startBit, int length, string plcId = null)
```

**使用示例：**
```csharp
// 读取I0.0开始的8个位
var iBits = await plcManager.ReadIBitsAsync(0, 0, 8);

// 读取I2.1开始的12个位
var iBits2 = await plcManager.ReadIBitsAsync(2, 1, 12);

// 指定PLC读取
var iBits3 = await plcManager.ReadIBitsAsync(0, 0, 32, "PLC2");
```

### 3. 读取M区不同类型数据

#### 3.1 读取M区各种数据类型

```csharp
/// <summary>
/// 读取M区不同类型数据
/// </summary>
/// <typeparam name="T">数据类型（bool, byte, short, int, float, double等）</typeparam>
/// <param name="startByte">起始字节</param>
/// <param name="count">读取数量</param>
/// <param name="plcId">PLC ID（可选）</param>
/// <returns>指定类型的数据数组</returns>
Task<T[]> ReadMAsync<T>(int startByte, int count = 1, string plcId = null)
```

**支持的数据类型：**
- `bool` - 布尔值
- `byte` - 字节（8位）
- `short` - 短整型（16位）
- `int` - 整型（32位）
- `float` - 单精度浮点数（32位）
- `double` - 双精度浮点数（64位）

**使用示例：**
```csharp
// 读取M0开始的1个字节
var mByte = await plcManager.ReadMAsync<byte>(0, 1);

// 读取M10开始的4个短整型
var mShorts = await plcManager.ReadMAsync<short>(10, 4);

// 读取M20开始的2个整型
var mInts = await plcManager.ReadMAsync<int>(20, 2);

// 读取M30开始的1个浮点数
var mFloat = await plcManager.ReadMAsync<float>(30, 1);

// 读取M40开始的3个双精度浮点数
var mDoubles = await plcManager.ReadMAsync<double>(40, 3);
```

#### 3.2 读取M区位数据

```csharp
/// <summary>

/// <param name="startByte">起始字节</param>

Task<bool[]> ReadMBitsAsync(int startByte, int startBit, int length, string plcId = null)
```

**使用示例：**
```csharp
// 读取M0.0开始的8个位
var mBits = await plcManager.ReadMBitsAsync(0, 0, 8);

// 读取M5.3开始的10个位
var mBits2 = await plcManager.ReadMBitsAsync(5, 3, 10);

// 指定PLC读取
var mBits3 = await plcManager.ReadMBitsAsync(0, 0, 64, "PLC1");
```

## 完整使用示例

```csharp
using S7NET.Services;
using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        var plcManager = MultiPlcServiceManager.Instance;
        
        try
        {
            // 初始化服务
            plcManager.Initialize();
            
            // 连接PLC
            await plcManager.ConnectAllPlcsAsync();
            
            // 1. 读取Q点数据
            Console.WriteLine("=== 读取Q点数据 ===");
            var qBits = await plcManager.ReadQBitsAsync(0, 0, 8);
            for (int i = 0; i < qBits.Length; i++)
            {
                Console.WriteLine($"Q0.{i}: {qBits[i]}");
            }
            
            // 2. 读取I点数据
            Console.WriteLine("\n=== 读取I点数据 ===");
            var iBits = await plcManager.ReadIBitsAsync(0, 0, 8);
            for (int i = 0; i < iBits.Length; i++)
            {
                Console.WriteLine($"I0.{i}: {iBits[i]}");
            }
            
            // 3. 读取M区不同类型数据
            Console.WriteLine("\n=== 读取M区数据 ===");
            
            // 读取字节数据
            var mBytes = await plcManager.ReadMAsync<byte>(0, 4);
            for (int i = 0; i < mBytes.Length; i++)
            {
                Console.WriteLine($"MB{i}: {mBytes[i]}");
            }
            
            // 读取整型数据
            var mInts = await plcManager.ReadMAsync<int>(10, 2);
            for (int i = 0; i < mInts.Length; i++)
            {
                Console.WriteLine($"MD{10 + i * 4}: {mInts[i]}");
            }
            
            // 读取浮点数据
            var mFloats = await plcManager.ReadMAsync<float>(20, 2);
            for (int i = 0; i < mFloats.Length; i++)
            {
                Console.WriteLine($"MD{20 + i * 4}: {mFloats[i]}");
            }
            
            // 读取M区位数据
            var mBits = await plcManager.ReadMBitsAsync(0, 0, 16);
            for (int i = 0; i < mBits.Length; i++)
            {
                Console.WriteLine($"M{i / 8}.{i % 8}: {mBits[i]}");
            }
            
        }
        catch (Exception ex)
        {
            Console.WriteLine($"操作失败: {ex.Message}");
        }
        finally
        {
            // 断开连接
            await plcManager.DisconnectAllPlcsAsync();
        }
    }
}
```

## 地址映射说明

### Q点地址映射
- `ReadQBitsAsync(0, 0, 8)` → Q0.0 到 Q0.7
- `ReadQBitsAsync(1, 3, 5)` → Q1.3 到 Q2.0 (跨字节)

### I点地址映射
- `ReadIBitsAsync(0, 0, 8)` → I0.0 到 I0.7
- `ReadIBitsAsync(2, 1, 12)` → I2.1 到 I3.4 (跨字节)

### M区地址映射
- `ReadMAsync<byte>(0, 1)` → MB0
- `ReadMAsync<short>(10, 1)` → MW10
- `ReadMAsync<int>(20, 1)` → MD20
- `ReadMAsync<float>(30, 1)` → MD30
- `ReadMBitsAsync(0, 0, 8)` → M0.0 到 M0.7

## 错误处理

所有方法都包含完整的错误处理机制：
- 连接断开自动检测
- 自动重连机制
- 详细的错误日志
- 友好的异常信息

```csharp
try
{
    var data = await plcManager.ReadMAsync<int>(0, 1);
}
catch (InvalidOperationException ex) when (ex.Message.Contains("PLC连接已断开"))
{
    Console.WriteLine("PLC连接断开，等待自动重连...");
    await Task.Delay(2000);
    // 可以选择重试
}
catch (Exception ex)
{
    Console.WriteLine($"读取失败: {ex.Message}");
}
```

## 注意事项

1. **数据类型对齐**: 确保读取的数据类型与PLC中的数据类型匹配
2. **地址范围**: 确保读取的地址在PLC的有效范围内
3. **连接状态**: 读取前确保PLC已连接
4. **异常处理**: 建议使用try-catch处理可能的异常
5. **性能考虑**: 批量读取比单个读取更高效
