# PLC自动断连和重连功能实现说明

## 功能概述

实现了智能的PLC连接管理功能：
1. **自动断连检测** - PLC断开时程序自动检测并更新状态
2. **自动重连机制** - PLC重新上线时程序自动重连
3. **优雅异常处理** - 连接断开时不抛出异常，优雅处理
4. **界面状态同步** - 实时更新界面显示连接状态

## 实现原理

### 1. 连接断开检测

#### ReadDBAsync方法优化
```csharp
catch (Exception ex)
{
    if (IsConnectionLostException(ex))
    {
        OnCommunicationError($"PLC连接已断开，正在尝试重连...");
        
        // 强制断开连接
        await ForceDisconnectAsync();
        
        // 触发连接状态变更事件
        OnConnectionStatusChanged(false);
        
        // 抛出友好异常
        throw new InvalidOperationException("PLC连接已断开", ex);
    }
}
```

**关键改进**：
- 检测到连接断开时立即强制断开连接
- 触发状态变更事件通知界面
- 抛出友好的异常信息

### 2. 自动重连机制

#### 心跳检测增强
```csharp
private void HeartbeatCheck(object state)
{
    var wasConnected = IsConnected;
    var currentlyConnected = IsConnected;

    if (wasConnected && !currentlyConnected)
    {
        // 连接断开了
        _lastDisconnectTime = DateTime.Now;
        _reconnectAttempts = 0;
        OnConnectionStatusChanged(false);
    }
    else if (!wasConnected && !currentlyConnected && _autoReconnectEnabled)
    {
        // 尝试自动重连
        TryAutoReconnect();
    }
}
```

#### 智能重连策略
```csharp
private void TryAutoReconnect()
{
    // 检查重连条件
    if (_reconnectAttempts >= _maxReconnectAttempts) return; // 最大3次
    if (timeSinceDisconnect.TotalSeconds < 5) return;       // 等待5秒
    if (_isConnecting) return;                              // 避免重复连接

    _reconnectAttempts++;
    
    // 异步执行重连
    Task.Run(async () =>
    {
        var success = await ConnectAsync(CancellationToken.None);
        if (success)
        {
            OnCommunicationError($"PLC {_ipAddress} 自动重连成功");
        }
    });
}
```

### 3. 界面状态管理

#### 数据读取优化
```csharp
catch (InvalidOperationException ex) when (ex.Message.Contains("PLC连接已断开"))
{
    // PLC连接已断开，更新状态但不停止定时器（等待自动重连）
    UpdatePLC1Data("连接断开", "等待重连...");
    // 不添加日志，避免日志过多
}
```

#### 连接状态事件处理
```csharp
private void OnConnectionStatusChanged(object sender, (string PlcId, bool IsConnected) args)
{
    var connectedCount = _multiPlcService.GetAllConnectionStatuses().Values.Count(v => v);
    if (connectedCount == 0)
    {
        UpdateStatus("所有PLC连接已断开，等待自动重连...");
        // 不停止定时读取，让它继续运行以便检测重连
    }
    else
    {
        UpdateStatus($"已连接 {connectedCount} 个PLC");
        // 如果有PLC重连成功，确保定时读取正在运行
        if (args.IsConnected && _readTimer == null)
        {
            StartPeriodicReading();
        }
    }
}
```

## 功能特性

### ✅ 自动断连检测
- **实时检测** - 通过异常捕获检测连接断开
- **强制断开** - 立即清理连接资源
- **状态更新** - 实时更新界面状态
- **友好提示** - 显示"连接断开，等待重连..."

### ✅ 智能自动重连
- **延迟重连** - 断开5秒后开始重连尝试
- **重连限制** - 最多尝试3次重连
- **异步重连** - 不阻塞主线程和界面
- **成功通知** - 重连成功时显示提示

### ✅ 优雅异常处理
- **不抛出异常** - 连接断开时不崩溃程序
- **继续运行** - 定时读取继续运行等待重连
- **日志优化** - 减少重复的错误日志
- **状态同步** - 界面状态与实际连接状态同步

### ✅ 用户体验优化
- **实时反馈** - 连接状态实时显示
- **自动恢复** - PLC重新上线时自动恢复数据读取
- **状态提示** - 清晰的状态信息："等待重连..."
- **无需干预** - 用户无需手动重连
