# S7NET 扩展读取功能实现总结

## 功能实现完成 ✅

按照您的要求，我已经成功添加了以下扩展读取功能：

### 1. 读取Q点（输出点）bool量 ✅
- **方法名**: `ReadQBitsAsync(int startByte, int startBit, int length)`
- **功能**: 读取指定起始位置和长度的Q点bool数据
- **支持**: 设置长度，跨字节读取

### 2. 读取I点（输入点）bool量 ✅
- **方法名**: `ReadIBitsAsync(int startByte, int startBit, int length)`
- **功能**: 读取指定起始位置和长度的I点bool数据
- **支持**: 设置长度，跨字节读取

### 3. 读取M区不同类型数据 ✅
- **方法名**: `ReadMAsync<T>(int startByte, int count)`
- **功能**: 读取M区各种数据类型（byte, short, int, float, double等）
- **支持**: 泛型类型，批量读取

### 4. 读取M区位数据 ✅
- **方法名**: `ReadMBitsAsync(int startByte, int startBit, int length)`
- **功能**: 读取M区指定位置和长度的位数据
- **支持**: 设置长度，跨字节读取

## 使用示例

### 基本用法
```csharp
var plcManager = MultiPlcServiceManager.Instance;

// 读取Q点数据
var qBits = await plcManager.ReadQBitsAsync(0, 0, 8);  // Q0.0-Q0.7

// 读取I点数据  
var iBits = await plcManager.ReadIBitsAsync(1, 3, 5);  // I1.3-I2.0

// 读取M区整型数据
var mInts = await plcManager.ReadMAsync<int>(10, 2);   // MD10, MD14

// 读取M区位数据
var mBits = await plcManager.ReadMBitsAsync(0, 0, 16); // M0.0-M1.7
```

### 多PLC环境
```csharp
// 指定PLC读取
var qData = await plcManager.ReadQBitsAsync(0, 0, 8, "PLC1");
var iData = await plcManager.ReadIBitsAsync(0, 0, 8, "PLC2");
var mData = await plcManager.ReadMAsync<float>(20, 1, "PLC1");
```

## 地址映射规则

### Q点地址映射
- `ReadQBitsAsync(0, 0, 8)` → Q0.0 到 Q0.7
- `ReadQBitsAsync(1, 3, 5)` → Q1.3 到 Q2.0 (跨字节)

### I点地址映射
- `ReadIBitsAsync(0, 0, 8)` → I0.0 到 I0.7
- `ReadIBitsAsync(2, 1, 12)` → I2.1 到 I3.4 (跨字节)

### M区地址映射
- `ReadMAsync<byte>(0, 1)` → MB0
- `ReadMAsync<short>(10, 1)` → MW10
- `ReadMAsync<int>(20, 1)` → MD20
- `ReadMAsync<float>(30, 1)` → MD30
- `ReadMBitsAsync(0, 0, 8)` → M0.0 到 M0.7
